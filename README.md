# Hotel Room Reservation System
Hotel Room Reservation System with floor-wise visualization, room booking, vacating, and reset operations built using HTML, CSS, and JavaScript. Developed for the Unstop SDE recruitment assessment.

## Features
✔ Visualization of hotel floors (10 → 1)  
✔ 97 rooms (Floors 1–9: 10 rooms, Floor 10: 7 rooms)  
✔ Book room by number  
✔ Random booking for available room  
✔ Vacate room via modal interaction  
✔ Reset to clear all bookings  
✔ No installation required (frontend-only)  

## Tech Stack
- HTML
- CSS
- JavaScript

## How to Run Locally
1. Clone the repository
2. Navigate to the `frontend` directory
3. Open `index.html` in your browser

## Unstop Assessment Context
This project was developed as part of the Unstop Software Development Engineer assessment, focusing on UI behavior, visualization, and booking interaction logic.
